# Artificial Intelligence Lab - Python Programs

A collection of Python programs developed as part of my **Artificial Intelligence Laboratory** coursework. This repository demonstrates fundamental concepts of data analysis, probability distributions, and data visualization using popular Python libraries.

## 📚 About

This repository contains **11 AI Lab programs** implemented in Python. The programs cover basic scientific computing and data visualization techniques using:

- NumPy
- Pandas
- Matplotlib

These exercises were completed as part of my undergraduate **Artificial Intelligence Lab**.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Lab-Python.git
```

Move into the project directory:

```bash
cd AI-Lab-Python
```

Install the required libraries:

```bash
pip install numpy pandas matplotlib
```

## ▶️ Running a Program

Run any program using:

```bash
python Program1.py
```

Replace `Program1.py` with the desired program.
